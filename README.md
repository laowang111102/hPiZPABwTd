# 电影院影片管理系统 python411

## 项目概述

本项目是基于Python开发的电影院影片管理系统，旨在为电影院提供一套便捷、高效的电影信息管理解决方案。

## 技术栈

- Python
- Django框架
- MySQL数据库
- HTML/CSS/JavaScript

## 功能模块

### 系统管理

- 用户管理
  - 用户注册
  - 用户登录
  - 用户权限控制
- 影院管理
  - 影院信息维护
  - 影厅管理

### 影片管理

- 影片信息维护
  - 影片新增
  - 影片修改
  - 影片删除
- 影片分类管理

### 排片管理

- 影片排期
- 影厅排期

## 系统角色

- 系统管理员
- 影院管理员
- 用户

## 运行环境

- Python 3.6+
- Django 2.2+
- MySQL 5.7+

## 部署步骤

1. 安装Python环境和相关依赖
2. 安装MySQL数据库
3. 克隆本项目代码
4. 导入数据库脚本
5. 修改配置文件
6. 运行项目
